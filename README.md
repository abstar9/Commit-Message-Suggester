Commit Message Suggester is an intelligent tool that helps developers write better commit messages. Simply describe your code changes, and the AI-powered system generates multiple conventional commit message suggestions following industry standards (feat, fix, docs, refactor, etc.).

Features:
• AI-powered suggestions using Google Gemini Pro
• Conventional commit format compliance
• Modern, responsive UI
• One-click copy to clipboard
• Type-safe with Pydantic and TypeScript

Tech Stack: FastAPI, Next.js 14, React 18, TypeScript, Google Gemini Pro
